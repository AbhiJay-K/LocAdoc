Frequently asked questions (FAQ)

1) Why am I not able to log in?

A: Please check if you have enabled network connectivity and entered valid password.


2) How to change password?

A: Go to settings on the main menu and click on password. Enter your current password and new password.


3) Is there a feature for forgotten password?

A: Yes, you can click to on forget password and you will receive a confirmation code via your registered email. Enter this confirmation code to reset your password.


4) What about offline access?

A: No this app requires network connection.


5) How do I lock or unlock document?

A: Move into the area where you have saved the document and click the folder icon at the bottom right corner. Click on the are name followed by the file name.


6) Who can lock or unlock document?

A: Only you can unlock the document


7) How do I know a document is locked?

A: If you are unble to view the document then it has been locked.


8) Where exactly are my documents hiding?

A: The documents in the device are stored in the hidden app folder and are encrypted


9) Can I restore the deleted document?

A: No if the files are deleted there is no way to recover.


10) I changed/am changing devices. How do I transfer my files?

LocAdoc comes with that includes 1GB of cloud storage.When you try to access document on your new device the app will prompt you to download the document from cloud storage. If you wish to download all backed up files, go to settings and click ‘download backup’.


11) What happens to my documents after i uninstall the app?

A: Uninstalling app will not remove the files from cloud. The files will be downloaded when the app is reinstalled.


12) What happens if I factory reset my device?

A: Factory reset will erase all files/data in device's internal storage. It is recommended to back up files using cloud backup before performing this reset.


13) How secure is this app?

A: Files will be secured using undistructable algorithms (AES 128).


13) How can I check the amount of space left in cloud?

A: Go to settings to view usage information.
